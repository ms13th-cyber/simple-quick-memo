ダッシュボードに、自分専用のシンプルなメモ帳を追加します。

== Description ==

Simple Quick Memoは、WordPressのダッシュボードにウィジェット形式のメモ機能を追加する軽量なプラグインです。

* ログイン後、すぐに備忘録やTODOリストを確認できます。
* データベースのオプション機能を利用しているため、動作が非常に高速です。
* 複雑な設定は一切不要。有効化するだけで使えます。

== Installation ==

1. `quick-memo` フォルダを `/wp-content/plugins/` ディレクトリにアップロードします。
2. WordPressの「プラグイン」メニューからプラグインを有効化します。
3. ダッシュボード（管理画面のトップ）に移動すると「クイックメモ」ウィジェットが表示されます。

== Frequently Asked Questions ==

= メモはどこに保存されますか？ =
WordPressの標準的な `options` テーブルに保存されます。

= 他のユーザーにもメモは見えますか？ =
現在のバージョンでは、管理画面にアクセスできる全ユーザーが同じメモを共有する仕様です。
